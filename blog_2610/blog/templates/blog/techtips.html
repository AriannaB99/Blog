Tech Tips

Command Description Example
git status

This git command is used to check if there have been any untracked changes have been made in your repository without being committed

 
(base) C:\Users\arian\CS2610-Bunnell-Arianna-Assn1> git status 
On branch master
Your branch is ahead of 'origin/master' by 26 commits. 
(use "git push" to publish your local commits)
git add

This git command is used to add either a file with untracked changes, or a previously untracked file to your git repository

 
(base) C:\Users\arian\CS2610-Bunnell-Arianna-Assn1> git add techtips.html
git commit

This git command is used to record changes which you have made to your local files after you have added them, taking a "snapshot" of the file at that point in time, adding a message to be paired with your commit in quotes after the -m 

(base) C:\Users\arian\CS2610-Bunnell-Arianna-Assn1> 
git commit techtips.html -m "sample commit"
[master c91722e] sample commit
1 file changed, 17 insertions(+)
git push
